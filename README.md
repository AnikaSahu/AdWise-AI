# AdWise AI 🎯
### AI-Powered Advertising Platform Recommendation System

> **Major Project | Session 2025-26 | Computer Science & Engineering**

---

## 📌 Overview

**AdWise AI** is a machine learning-based web application that helps businesses choose the **most effective advertising platform** for their campaigns. Instead of guessing or wasting budget on trial-and-error, AdWise AI analyzes key business inputs and delivers **data-driven recommendations with probability scores**.

---

## ❗ Problem Statement

* **Businesses lose significant ad budget** due to poor platform selection
* Most decisions are based on **assumptions, not data**
* Platforms like *Facebook, Instagram, Google Ads, and YouTube* perform differently based on:
  * Industry type
  * Budget range
  * Target audience
  * Campaign objective
* ❌ Wasted advertising budget
* ❌ Poor campaign performance
* ❌ Low return on investment (ROI)

---

## 💡 Our Solution

AdWise AI takes **4 key business inputs** and predicts the best-fit advertising platform:

| Input | Description |
|-------|-------------|
| 🏭 Industry | Business sector / domain |
| 💰 Budget | Advertising spend range |
| 👥 Target Audience | Demographics & user segments |
| 🎯 Campaign Goal | Awareness / Leads / Conversions |

---

## ⚙️ Tech Stack & Approach

* **Data Collection** — Structured dataset with 4 core business features
* **Data Preprocessing** — Cleaning & encoding for ML compatibility
* **Model Development** — Trained using **Random Forest Classifier**
* **Backend Integration** — ML model served via **REST API**
* **Frontend Dashboard** — Built with **EJS** (form input + visual charts)
* **Admin Panel** — Tracks user activity and prediction history

---

## 📊 Key Highlights

* ✅ **1 ML Model** — Random Forest for multi-class platform classification
* ✅ **4 Advertising Platforms** covered — Facebook, Instagram, Google Ads, YouTube
* ✅ **Probability Scores** displayed for every recommendation
* ✅ **Admin Dashboard** for monitoring all predictions
* ✅ **Visual Charts** for easy result interpretation

---
